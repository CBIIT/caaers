Adverse Events

  1. Evaluation Period: Click the drop down list to select an existing evaluation period. To create a new evaluation period, select Create New from the drop down list.
  2. If you select Create New an Evaluation Period Details form will appear. Complete the fields with appropriate information.
    Note:     All fields with a red asterisk (    *) are required.
  3. Click Save when you are done.
  4. If you have completed all of the required fields correctly you will see a Confirmation page. Click OK.
  5. The full Adverse Event pages opens with sections for evaluation period details, observed adverse events, and solicited adverse events.
    Note:     All fields with a red asterisk (    *) are required.
    • To edit the evaluation period details, click Edit. The Evaluation Period Details form will appear. Complete the fields with appropriate information and click Save.
    • To add a single adverse event, type in the adverse event, select it from the drop down list and click Add.
    • To add multiple adverse events, click Add Multiple
      Note:       This page will look different based on the vocabulary chosen for the study (CTC or MedDRA). Select the AEs to enter and click Add.
    • Enter details for each adverse event you've added, entering all required information and
    • Enter details for the solicited adverse events. Based on your response to Grade, other fields may be required.
      Note:       The Verbatim text boxes are for notes from the medical chart.
  6. Click Save to Save the changes, or Save & Continue to determine if reporting is required.
