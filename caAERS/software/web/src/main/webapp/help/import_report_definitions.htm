Import Report Definitions

Click the Rules tab and click Import Report Definitions in the task menu.

To Import a Report Definition in XML Format:

  1. Click Browse and locate the XML file for the report definition. To view an example of a report definition in XML format, click here.
  2. Click Import. If the import was successful you will see a message that confirms it has been imported.
  3. Click List Report Definitions in the Rules tab menu and the imported report definition should now appear in the list.
