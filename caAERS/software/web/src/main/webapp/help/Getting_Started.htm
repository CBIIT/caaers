Getting Started

Navigation elements of caAERS are found at the top of the page. The browser navigation elements should not be used instead of the tabs and buttons on the caAERS web screens. Using the forward and back browser navigation elements may cause problems with the system and could cause you to lose information if you are in the middle of entering a study or AE. The application contains all necessary buttons and navigation elements in the web screens of the application.

 

These include the Navigation Menu (tabs at the top), the Task menu (middle row of links), and the steps menu (bottom arrows). Each page will also contain buttons to help navigate through the tasks.
