Patient Disease Information

Disease Name: 

The list of values for disease names displays MedDRA (Medical Dictionary for Regulatory Activities) terminology.  The International Conference on Harmonization (ICH) develops requirements for drug regulatory reporting globally and has chosen MedDRA terminology to be the standard.  To facilitate data transfer to the FDA and other regulatory agencies, CTEP mapped wherever possible, caAERS lists of values to MedDRA.  Many disease names include "NOS" (not otherwise specified) providing a reportable general term for a wide range of related diseases.

The Disease Name list is best approached by first identifying the Category of disease (i.e., Endocrine, Gastrointestinal, etc) followed by review of the diseases within each Category.  caAERS does not require specific details regarding Disease Name.

Unable to find Disease Name on the list?

Select either Solid Tumor NOS or Hematologic unspecified from the list of values AND enter the Disease Name in the "Disease Name Not Listed" field.

If the Primary Site of Disease is unknown, enter “unknown” in the “Other Primary Site of Disease” field.

Section # 5: Treatment Information

Treatment

First course start date:

Enter the date the first course (or cycle) began that is associated with the Adverse Event or death being reported.  Enter the first date of administration of the first agent to the patient.

Adverse event course start date:

Enter the date the course (or cycle) began that is associated with the Adverse Event or death for this report. Some protocols do not have defined courses or cycles, such as those with daily administration of an agent.  For these protocols “Adverse event course start date” is the date the patient last received the agent prior to experiencing the event or death.

Course number on which the event occurred: 

Enter the course (or cycle) associated with the Adverse Event(s) or death for this report. Some protocols do not have defined courses or cycles, such as those with daily administration of an agent.  For these protocols “Course number on which event occurred” is to be entered as 1 (one).

Section #10: Attribution for Adverse Events

Attribution

Every caAERS report that includes Adverse Events must include for each Adverse Event at least one attribution of Possible, Probable, or Definite to either the Agent, the Disease, Other Causes, or Concomitant Medications. The physician is to indicate at least a possible causal relationship for each Adverse Event.

All Adverse Events (AEs) entered in both the Assessment section and the Adverse Event section appear. Each AE is evaluated to determine what might have caused the event or what interventions or conditions to which the event might have been associated. Evaluation of AEs involves assessing the relationship of the event to:

•      Investigational Agents

•      Concomitant Medications

•      Other Causes

Agent and disease information is mandatory in Expedited Reports. Therefore agent and disease information must be entered, saved and addressed in the

“Attribution for Adverse Event” section before a report can be submitted. “Concomitant Medications” and “Other Causes” are not applicable for all patients. Therefore if Concomitant Medications and Other Causes were not selected as appropriate for this report, only the Protocol Agents and the Disease will appear on the table and require assignment of attribution. However, if Concomitant Medications and Other Causes were selected as appropriate for this report, Protocol Agents, Disease, Concomitant Medications and Other Causes will appear on the table and require assignment of attribution

Adverse Events appear as column headers on the top of the table including the AE level, grade, and disease.  Concomitant Medications and/or Other Causes appear on the left of the table if applicable. Complete this section by addressing each event one at a time:

Course:

Was (the first event) related to the Course?  Choose from the drop down List of Values: Unrelated, Unlikely, Possible, Probable, Definite.

Concomitant medication:

Was (the first event) related to the Concomitant medication(s)?  Choose from the drop down List of Values: Unrelated, Unlikely, Possible, Probable, Definite.

Other cause:

Was (the first event) related to Other Causes? Choose from the drop down List of Values: Unrelated, Unlikely, Possible, Probable, Definite.

o  Repeat the steps for the Second Event and all subsequent events if applicable. Click on Update or Continue when the Attributions of Event section is completely filled. If any data fields are entered incorrectly, an appropriate Error message will appear after Update or Continue is clicked. Error messages will describe data problems in specific fields. Resolve the error and correct the data, and click Update or Continue.
