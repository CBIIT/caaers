Enter Subject - Details

The details section is where you add general information on the subject and add an identifier.

  1. Select the Site from the drop down box.
  2. Enter First Name.
  3. Enter Last Name.
  4. Enter Maiden Name if desired.
  5. Enter Middle Name if desired.
  6. Enter Date of Birth yyyy mm dd) Optionally, you can enter just the year of birth and not the full birth date
  7. Select Gender.
  8. Select Ethnicity.
  9. Select Race.
  10. Determine if you are entering a System Identifier or Organzation Identifier (caAERS automatically displays the Organization Identified fields). If you are entering an Organization Identifier, enter the Identifier, select Identifier Type, and enter Organization. Select Primary Indicator if necessary.
    OR
    If you are entering a System Identifier, click the red X next to     Organization Identifier and click Add System Identifier. Enter the Identifier, select Identifier Type, and enter the System name. Select Primary Indicator if necessary.
  11. If there is more than one Identifier, click Add System Identifier or Add Organization Identifier and repeat step 10.
  12. Select Primary Indicator for only one Identifier, and then click Continue to select a study.
