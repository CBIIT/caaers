Welcome

The Cancer Adverse Event Reporting System (caAERS) is a Web application that can be used for documenting, managing, reporting, and analyzing adverse events (AEs). caAERS is a repository for capturing and tracking routine and serious AEs as well as a tool for preparing and submitting expedited AE reports to regulatory agencies.
