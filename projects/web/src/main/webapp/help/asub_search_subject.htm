Assign Subjects to Studies - Search Subject

You can add a subject to another study in caAERS at any time using the Assign Subjects to Studies task.
 

Note: This method allows assignment to one study at a time, so if there are multiple studies the subject needs to be added to, you will repeat this task.

 

Search for a Subject

    1. Select Subject criteria.
    2. Enter Search criteria and click Search.
    3. Select the subject from the results and click Continue.
