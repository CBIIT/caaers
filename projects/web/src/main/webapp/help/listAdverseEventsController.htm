Manage Reports

The Manage reports task of the Adverse Event tab is a centralized reporting area where you can view a list of all reports, both expedited and routine, for a given user-study combination. You can also see if the report is incomplete, select the report to view the full details, generate a pdf or xml version of the report, generate a MedWatch pdf, or send notification to the patient study calendar.

To manage a subject’s reports for a specific study:

  1. Click the Adverse Events tab in the navigation bar and Enter the first few letters of the subject’s name in the Select subject field and select the subject’s name from the drop down list that appears.
  2. Enter the first few letters of the name of the study in the Select study field and select it from the drop down menu that appears.
  3. Click Continue.
    Note: You can select the study first &endash; the order does not matter
  4. At the next page you can view all the reports. Click on the Evaluation Period to view and/or edit the report.
 

Note: There may be additional actions based on the report selected.
