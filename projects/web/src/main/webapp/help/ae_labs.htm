Labs

If tests were sent to a lab and are possibly related to the AE, include them in the report.

To add a lab:
 

  1. Click Add a lab.
  2. Select the Lab category from the drop down box.
  3. Select the Lab test name from the drop down box. If the lab test is not listed, select Other, specify and enter the lab test name in the Other field.
    Note:     If you selected Microbiology for the Lab category, go to step 8.
  4. Select the Units from the drop down box that will be associated with the Baseline, Worst, and Recovery values.
  5. Enter Baseline value and date (mm/dd/yyyy).
  6. Enter Worst value and date (mm/dd/yyyy).
  7. Enter Recovery value and date (mm/dd/yyyy), if applicable and go to step 11.
  8. Enter the Site associated with the lab test.
  9. Enter the Date of the test.
  10. Enter details for the Infectious Agent.
  11. To add another lab, repeat steps 1-10.
  12. Click Save & Continue.

Note: To delete lab information, click the X in the right corner of the window.
